# Crashmap 360:  A data-driven infrastructure recommendation system for Washington, D.C.
This repository includes all the code utilized in the original article "Crashmap 360:  A data-driven infrastructure recommendation system for Washington, D.C."
The assistance of large language models was used in developing parts of Crashmap 360's software. All design, algorithms, and methodology decisions, along with interpretations, were made by the author.
